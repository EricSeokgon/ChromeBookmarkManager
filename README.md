# Chrome Bookmark Manager

Organize your bookmarks: remove duplicates, clean empty folders, and check dead links.

## Features

- **Remove Duplicates**: Find and remove duplicate bookmarks to keep your list clean.
- **Clean Empty Folders**: Automatically detect and delete folders that contain no bookmarks.
- **Check Dead Links**: Scan your bookmarks for broken URLs (404s, DNS errors, etc.).

## Installation

1.  Clone this repository or download the source code.
2.  Open Google Chrome and navigate to `chrome://extensions/`.
3.  Toggle **Developer mode** in the top right corner.
4.  Click **Load unpacked** in the top left.
5.  Select the directory containing this project (where `manifest.json` is located).

## Usage

1.  Click the **Chrome Bookmark Manager** icon in your browser toolbar.
2.  Use the popup interface to access the manager features.
